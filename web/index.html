Smart Customer Support

Functionalities

  • Smart agent assignment using expertise, solved count, review score, and workload.
  • Priority scoring from severity prediction.
  • Feedback-aware score updates for closed tickets.
  • Exportable assignment output for dashboards and reporting.

Run python3 app.py --export data/processed_tickets.json to generate data for UI integration.
